# Personal Portfolio Website

An interactive, animated portfolio website featuring live CSS coding animation. This project showcases personal information, professional experience, and projects through an engaging typing animation effect.

## Features

- **Live CSS Animation**: Watch CSS code being typed out character by character with real-time syntax highlighting
- **Interactive Work Section**: Flip between markdown and rendered HTML views
- **3D Perspective Transforms**: Modern CSS 3D effects for an immersive experience
- **Editable CSS**: Users can edit CSS directly in the browser
- **Responsive Design**: Works across different screen sizes and browsers

## Getting Started

### Prerequisites

- Node.js (v14 or higher recommended)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/dohaidang/dohaidang-introduce.git
cd dohaidang-introduce

# Install dependencies
npm install
```

### Development

```bash
# Start the development server
npm run dev

# Open http://localhost:4003/index-dev.html in your browser
```

### Production Build

```bash
# Build for production
npm run build

# The built files will be in the /dist directory
```

## Tech Stack

- **JavaScript (ES6+)** with Babel for transpilation
- **Webpack** for bundling and module management
- **Markdown** library for content rendering
- **Bluebird** for promise handling
- **CSS3** with 3D transforms and animations

## Customization

To personalize this portfolio:

1. Update `work.txt` with your personal information, experience, and projects
2. Modify `styles0.css` through `styles3.css` to customize the animation sequence
3. Update `header.html` with your social links
4. Edit `index.html` and `index-dev.html` to change the page title
